# ML_TimeSeries_Bike

## Bike Rental Prediction using Deep Learning: Time Series

Este projeto foi desenvolvido como parte do Curso de Deep Learning para Séries Temporais oferecido pela Alura (https://cursos.alura.com.br/course/deep-learning-previsao-keras). O objetivo principal é prever a quantidade de bicicletas que serão alugadas em um determinado horário no futuro, utilizando técnicas avançadas de aprendizado profundo (Deep Learning) aplicadas a séries temporais.

Descrição do Projeto
O conjunto de dados utilizado neste projeto contém a contagem de bicicletas alugadas em diferentes horários, e nosso desafio é construir um modelo capaz de realizar previsões precisas. Técnicas como Redes Neurais Recorrentes (RNNs), LSTM e GRU são empregadas para a previsão de séries temporais.
